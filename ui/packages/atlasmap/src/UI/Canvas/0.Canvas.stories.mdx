# Canvas

Renders a resizable SVG which allows panning and zooming the content. Try dragging around the example, or scrolling
mouse wheel.

<Story name="Example">{stories.example()}</Story>

A `Canvas` component must be used inside a `CanvasProvider` component. If `width` or `height` are not provided to the
`CanvasProvider`, they will be automatically determined.
