# Arc

Elements rendered inside a <LinkTo kind='canvas/canvas' story='example'>Canvas</LinkTo> can be linked together using a
`Arc`.

A `Arc` is rendered as an arch between a start and and end point, specified as SVG coordinates. The width and the
color of the arch can be customized.
