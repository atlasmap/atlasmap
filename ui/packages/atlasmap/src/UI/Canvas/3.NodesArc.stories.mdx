# NodesArc

It's possible to create <LinkTo kind='canvas/arc' story='example'>Arc</LinkTo>s between elements without knowing the
start and end coordinates beforehand using `NodesArc` and `NodeRef`.

This enables the possibility to move elements inside a `Canvas` programmatically without having to deal with coordinates
change. For example, nodes could be animated:
